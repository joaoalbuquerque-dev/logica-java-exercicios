# 📌 Vetores

Exercícios de lógica em Java envolvendo vetores.

📚 Fonte: Beecrowd + Curso Nélio Alves

## 📚 Exercícios resolvidos

- maior valor e posição no vetor
- números pares e contagem de pares
- soma de dois vetores (A + B)
- média dos elementos do vetor
- elementos abaixo da média
- média dos números pares
- pessoa mais velha (nome e idade)
- alunos aprovados (média >= 6)
- análise de alturas (maior, menor, média, quantidade)
- controle de lucro de mercadorias

## 🧠 O que foi praticado

- leitura de vetores
- percorrer vetor com `for`
- comparação de valores
- uso de acumuladores e contadores
- operações com múltiplos vetores
- lógica com dados combinados (nome + idade, notas, etc.)
