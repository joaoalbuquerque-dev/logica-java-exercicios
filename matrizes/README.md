# 📌 Matrizes

Exercícios de lógica em Java envolvendo matrizes.

📚 Fonte: Curso Nélio Alves

## 📚 Exercícios resolvidos

- mostrar os valores negativos da matriz
- soma dos elementos de cada linha
- maior elemento de cada linha
- soma dos elementos acima da diagonal principal
- soma de duas matrizes
- leitura de linha, coluna, diagonal principal e alteração de valores negativos
- rotação/deslocamento de uma fila da matriz

## 🧠 O que foi praticado

- leitura de matrizes
- percorrer linhas e colunas com `for`
- acesso por índices `[i][j]`
- soma de elementos
- busca de maior valor
- operações com diagonal principal
- combinação de duas matrizes
- manipulação e alteração de valores
